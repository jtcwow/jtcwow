== Description ==

WooCommerce Multi Vendor Shipping Addon is a one stop WooCommerce shipping solution for your WooCommerce Multi-Vendor store. The plugin allows vendors to set up and manage real-time shipping rates from carriers like UPS, USPS, Stamps, DHL and FedEx.

WooCommerce Multi Vendor Shipping Addon requires you to install any one of the following WooCommerce shipping plugins by PluginHive:

* WooCommerce FedEx Shipping Plugin
* WooCommerce UPS Shipping Plugin
* WooCommerce Multi-Carrier Shipping Plugin

And one of the supported WooCommerce Multi Vendor plugins:

* Dokan Multi-Vendor Plugin for WooCommerce
* WooCommerce Product Vendors
* WCFM Marketplace
* WC Vendors Pro
* WC Marketplace

= How does it work? =

This addon transforms your WooCommerce online store into a powerful, robust and feature-rich online shopping centre with multiple independent vendors selling their products via a centralised product catalog and facilitates you to ship through different shipping carriers like UPS, FedEx, USPS, Stamps and DHL.

<ul>
<li>Allow your customers to buy products from multiple vendors in a single order.</li>
<li>Generate labels for separate vendors in an order.</li>
</ul>

1. Upload the plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Thats it! you can now configure the plugin.

== Frequently Asked Questions ==

= Which shipping carriers are supported by your Multi Vendor Shipping plugin? =

The plugin supports the following shipping carriers:
- FedEx
- UPS
- USPS
- Stamps
- DHL Express

= I am confused! What do I need to get started? =

To start using the multi-vendor functionality, you need to have :
- This addon
- Our FedEx / UPS / MultiCarrier Shipping plugin
- Supported multi-vendor plugins to capture the vendors(as mentioned in product page)

= Is the plugin configuration complicated? =

The plugin is very easy to configure. We have a step by step tutorial on setting up this addon. Our Help Desk also has extensive documentation which includes FAQs, Troubleshooting Guide, Knowledge Base and Code snippets. 

= Does the multi-vendor plugin support Pickup for vendors? =

Pickup is a feature that is specific to a carrier. Our FedEx and UPS plugins provides Pickup facility. However, this feature is restricted to requesting a pickup for the shop-owner. Pickup cannot be raised for Vendors.
